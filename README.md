# HireIn

> Find. Hire. Get It Done.

HireIn is a MERN-stack based on-demand local services marketplace that connects customers with trusted professionals such as electricians, plumbers, cleaners, painters, technicians, and more.

---

## 📌 Project Overview

HireIn was created to solve the difficulty of finding reliable local workers for everyday services. The platform allows users to search, compare, and hire professionals based on ratings, pricing, experience, and location.

To maintain security and platform trust, personal contact information remains hidden until a secure booking is made.

---

## 🚀 Features

### Customer Features
- Search nearby professionals
- Filter by category, ratings, pricing, and location
- View profiles and reviews
- Secure booking system
- In-app communication
- Booking management

### Service Provider Features
- Create professional profiles
- Add services and pricing
- Manage bookings
- Update availability
- Receive ratings and reviews

### Admin Features
- Manage users and professionals
- Verify service providers
- Monitor bookings and payments
- Manage platform commissions
- Handle complaints and reports

---

## 💳 Payment Model

HireIn uses a hybrid payment system:
- Customers pay a small booking fee through the platform
- Remaining payment is paid directly to the service provider after job completion

This model ensures:
- Secure platform commission
- Reduced fake bookings
- Better customer trust
